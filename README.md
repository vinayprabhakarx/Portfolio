# Portfolio Website

A modern, responsive portfolio website built with ReactJS 18, featuring a beautiful UI and seamless user experience.

## Live Demo

Visit [https://vinayprabhakar.tech](https://vinayprabhakar.tech)

## Features

- Modern and responsive design
- Contact form with email integration(now not working)
- Project showcase with filtering
- Dark/Light theme support
- SEO optimized

## Tech Stack

### Frontend
- React 18
- React Router 6
- Material-UI

### DevOps
- DigitalOcean
- Cloudflare
- GitHub
- PM2
- Nginx

## Project Structure

```
portfolio-react/
├── src/                  # Frontend source files
│   ├── components/       # React components
│   ├── pages/           # Page components
│   ├── hooks/           # Custom React hooks
│   ├── utils/           # Utility functions
│   └── styles/          # CSS and style files
├── public/             # Static files
```
