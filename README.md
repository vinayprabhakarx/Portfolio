# Portfolio Website

A sleek and modern portfolio website built with ReactJS 18, delivering an engaging and responsive user experience.

## Live Demo

Explore the website here: [https://vinayprabhakar.tech](https://vinayprabhakar.tech)

## Features

- Fully responsive design for seamless viewing on all devices  
- Contact form with email integration
- Blog (currently under maintenance)  
- Dynamic project showcase with filtering options  
- Dark/Light theme toggle for improved accessibility   

## Tech Stack

- **Frontend:** React 18, React Router 6, Material-UI  
- **Backend:** MailGun API for handling contact form emails  

### DevOps Tools

- Ubuntu  
- Cloudflare  
- GitHub  
- PM2  
- Nginx  

## Project Structure

